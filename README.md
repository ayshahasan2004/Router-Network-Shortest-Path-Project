Router Network Shortest Path Project

This project simulates a router network using graph algorithms in C.  
It implements both BFS and Dijkstra’s algorithm to find optimal paths.  
The network is represented using an adjacency matrix loaded from a file.  
Users can input a source and destination router through a menu system.  
BFS is used to find the shortest path in terms of hops.  
Dijkstra’s algorithm is used to find the minimum cost path.  
The program supports up to 20 routers in the network.  
It uses a Min-Heap structure to optimize Dijkstra’s performance.  
Dynamic memory allocation is used to store and return computed paths.  
Results include both the path and total cost between routers.  
Outputs can be saved to a text file for later analysis.  
The system handles invalid inputs and missing routes safely.  
Case-insensitive router input improves usability.  
This project demonstrates graph theory and data structures in C.  
It strengthens understanding of BFS, Dijkstra, and memory management.
